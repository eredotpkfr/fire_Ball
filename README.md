### HOW TO USE ?
1. First Download Script.<br/>
2. Install All Requirement Modules(I Wrote "requirements.txt").<br/>
3. Finally Use It.

#### BEFORE RUN
`root@kali:~/Desktop/fire_Ball# pip3 install -r requirements.txt`

#### RUN
`root@kali:~/Desktop/fire_Ball# python3 fire_Ball.py`
